# Ready-To-Drive Sound System (RTDS)

## Overview

The Ready-To-Drive Sound System (RTDS) is a safety subsystem that generates an audible warning signal before vehicle movement, alerting nearby personnel that the vehicle is ready for operation.

## Project Scope

This project focused on the design of the RTDS circuit and PCB layout for use in a Formula Student Electric vehicle.

## Responsibilities

- Circuit design
- Audio generation circuitry development
- Component selection
- Schematic capture
- PCB layout design
- Design documentation

## Skills Demonstrated

- Electronic Circuit Design
- PCB Design
- Audio Signal Generation
- Component Selection
- Safety System Development

## Current Status

Design phase completed. Manufacturing, assembly, and vehicle-level testing have not yet been performed.

## Project Context

Developed as part of Formula Student Electric vehicle development.
