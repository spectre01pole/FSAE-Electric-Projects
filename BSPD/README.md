# Brake System Plausibility Device (BSPD)

## Overview

The Brake System Plausibility Device (BSPD) is a safety subsystem used in Formula Student Electric vehicles to detect implausible driving conditions, such as simultaneous braking and acceleration requests, and initiate protective actions when required.

## Project Scope

This project focused on the design of the BSPD circuit and PCB layout to support Formula Student Electric safety requirements.

## Responsibilities

- Circuit design
- Signal monitoring circuitry development
- Component selection
- Schematic capture
- PCB layout design
- Design documentation

## Skills Demonstrated

- Analog Electronics
- Signal Conditioning
- PCB Design
- Safety System Design
- Hardware Documentation

## Current Status

Design phase completed. Manufacturing, assembly, and vehicle-level testing have not yet been performed.

## Project Context

Developed as part of Formula Student Electric vehicle development.
