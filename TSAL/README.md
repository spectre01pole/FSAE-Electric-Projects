# Tractive System Active Light (TSAL)

## Overview

The Tractive System Active Light (TSAL) is a safety-critical subsystem used in Formula Student Electric vehicles to indicate the status of the tractive system and provide a visual indication of high-voltage activity.

## Project Scope

This project focused on the design of the TSAL circuit and PCB layout in accordance with Formula Student Electric requirements.

## Responsibilities

- Circuit design
- Component selection
- Schematic capture
- PCB layout design
- Design documentation

## Skills Demonstrated

- Analog Electronics
- PCB Design
- Schematic Design
- Component Selection
- Safety-Critical System Design

## Current Status

Design phase completed. Manufacturing, assembly, and vehicle-level testing have not yet been performed.

## Project Context

Developed as part of Formula Student Electric vehicle development.
