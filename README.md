# FSAE Electric Projects

Projects developed as part of Formula Student Electric vehicle development.

## Developed Circuit Boards

### Tractive System Active Light (TSAL)
- Designed and tested the TSAL board.
- Used to indicate the status of the tractive system according to Formula Student regulations.

### Voltage Indicator Board
- Designed circuitry for monitoring and indicating system voltage levels.
- Ensures safe operation and easy diagnostics.

### Brake System Plausibility Device (BSPD)
- Developed the BSPD circuit for safety compliance.
- Detects faults and disconnects power when necessary.

### Ready-To-Drive Sound Board (RTDS)
- Designed the circuit responsible for generating the Ready-To-Drive sound.
- Provides audible indication before vehicle movement.

## Skills Applied
- Analog Circuit Design
- PCB Design
- Schematic Capture
- Component Selection
- Testing and Debugging
- Electrical Safety Systems
- Formula Student EV Regulations

These projects were developed during my involvement with the Formula Student Electric team.
