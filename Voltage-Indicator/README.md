# Voltage Indicator Board

## Overview

The Voltage Indicator Board is designed to provide visual indication of electrical system voltage conditions, supporting system monitoring and electrical safety awareness.

## Project Scope

This project focused on the design of the voltage indicator circuitry and PCB layout for Formula Student Electric applications.

## Responsibilities

- Circuit design
- Voltage indication circuitry development
- Component selection
- Schematic capture
- PCB layout design
- Design documentation

## Skills Demonstrated

- Analog Electronics
- PCB Design
- Voltage Monitoring Systems
- Schematic Design
- Hardware Documentation

## Current Status

Design phase completed. Manufacturing, assembly, and vehicle-level testing have not yet been performed.

## Project Context

Developed as part of Formula Student Electric vehicle development.
